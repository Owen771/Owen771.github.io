Google SSO - Strong Consistency in Practice

  • https://www.youtube.com/watch?v=aay7CzkL_u4

one-copy semantics refers to a consistency model where all processes accessing or updating a given file see the same version of the file, as if there were only a single copy of the file.
單一副本更新語意是指一種一致性模型，其中存取或更新給定檔案的所有程序都會看到該檔案的相同版本，就好像該檔案只有一個副本一樣

  • 一致性：所有行程看到的文件內容都是一致的，文件只有一份。這意味著一個進程所做的任何更新對於存取該檔案的所有其他進程都立即可見。
  • 可見性：一位客戶所做的更改對所有其他客戶立即可見。這可確保所有用戶端都能看到檔案的最新版本，從而保持分散式系統中的資料一致性。
  • 實作：這種語意通常可以在檔案有中央權限或單點事實的系統中實現，確保所有更新在所有客戶端上統一反映。

e.g After u changed ur password, you shld not still able to login with ur old password

  • against eventual consistency
  • NEED strong consistency
